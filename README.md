# SaaS Landing Page Project

This is a complete frontend project structure for a responsive SaaS landing page.

## Technologies Used
- HTML5
- CSS3 (Vanilla)
- JavaScript (Vanilla)

## Folder Structure & File Purposes

- **`index.html`**: The main entry point of the website. It contains the structural semantic HTML (Header, Main, Footer, Sections) and links the necessary CSS and JavaScript files.
- **`css/`**: Directory containing all stylesheets.
  - **`style.css`**: The core stylesheet for global variables, typography, colors, resets, and the base design of components.
  - **`responsive.css`**: Contains media queries used to adjust layouts for tablets and mobile devices. Kept separate for better maintainability.
- **`js/`**: Directory containing all JavaScript files.
  - **`script.js`**: The main script file containing interactivity logic, such as smooth scrolling, mobile menu toggles, and form validation.
- **`assets/`**: Directory for static resources.
  - **`images/`**: Stores raster images (JPG, PNG, WebP) used across the project like hero graphics and team photos.
  - **`icons/`**: Stores vector graphics (SVG) and icon assets used in features or social links.
- **`README.md`**: This file. Provides project documentation, setup instructions, and an overview of the architecture.

## Setup
Simply open `index.html` in your browser to view the project locally.
